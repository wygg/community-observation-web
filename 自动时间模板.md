# Obsidian 快速创建模板使用指南

## 🎯 一键创建新观察的方法

### 配置自动替换

#### 步骤1：设置快速替换
1. **Obsidian 设置** → **热键**
2. **设置命令**：`Editor: Find and replace` 设置为 `Ctrl/Cmd + R`

#### 步骤2：使用模板
1. **复制快速创建模板内容**
2. **粘贴到新文件**
3. **按 `Ctrl/Cmd + R` 打开替换**
4. **逐项替换**：

```
$1 → 你的观察标题
$2 → 2024-02-09 (自动填写)
$3 → 观察地点
$4 → 分类
$5 → ["标签1", "标签2", "标签3"]
$6 → 天气状况
$7 → 外观描述内容
$8 → 功能分析内容
$9 → 人文观察内容
$10 → 发现的问题
$11 → 改进建议
$12 → 总结感悟
```
