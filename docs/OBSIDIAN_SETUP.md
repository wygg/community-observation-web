# Obsidian 模板配置完成指南

## 🎯 配置完成后的使用流程

### 1. 快速创建新观察

#### 方法A：使用基础模板
1. **打开 Obsidian**
2. **在 `content/observations/` 文件夹中右键**
3. **选择"新建文件"**
4. **复制 `考现学观察模板.md` 内容**
5. **手动填写模板中的变量**

#### 方法B：使用智能模板（推荐）
1. **确保安装了 Templater 插件**
2. **在 `content/observations/` 文件夹中**
3. **按 `Ctrl/Cmd + N`**
4. **选择模板"智能观察模板"**
5. **按提示填写信息**

### 2. 插件推荐安装

#### 必装插件：
1. **Templates** - 基础模板功能
2. **Image auto upload** - 图片自动上传
3. **Tag Wrangler** - 标签管理
4. **File Explorer** - 文件管理

#### 可选插件：
1. **Templater** - 智能模板变量
2. **Calendar** - 日历视图
3. **Dataview** - 数据查询
4. **Excalidraw** - 手绘图表

### 3. 模板使用演示

#### 创建新观察步骤：
```
1. 进入 content/observations/ 文件夹
2. 使用模板创建文件：2024-02-07-垃圾分类观察.md
3. 系统会提示输入：
   - 观察地点：小区东门垃圾站
   - 天气状况：晴朗
   - 外观特征：4个分类垃圾桶，标识清晰...
   - 功能分析：分类明确，容量充足...
   - 问题：厨余垃圾桶容量不足...
   - 建议：增加厨余垃圾收集频次...
4. 保存文件
5. 添加图片到 images/observations/
6. Git 提交并推送
```

### 4. 常用标签预设

#### 建议的标签分类：
```markdown
# 分类标签
#公共空间 #交通设施 #绿化环境 #环保设施 #安全设施

# 特征标签  
#人性化 #设计 #便利 #安全 #美观 #实用性

# 评价标签
#推荐 #问题 #建议 #改进 #优化
```

### 5. 快捷操作设置

#### 创建快捷命令：
1. **设置** → **快捷键**
2. **设置模板快捷键**：`Ctrl/Cmd + Shift + T`
3. **设置新建文件快捷键**：`Ctrl/Cmd + Shift + N`

#### 工作区配置：
1. **保存工作区**：左侧菜单 → 工作区 → 保存
2. **创建专用工作区**：只显示 content/observations/ 和 images/observations/

### 6. 图片管理优化

#### 设置图片默认路径：
1. **设置** → **文件与链接** → **默认附件路径**
2. **设置为**：`images/observations/`
3. **新建附件时自动放入正确文件夹**

#### 图片重命名规则：
```
原文件：IMG_20240207_001.jpg
重命名：garbage-sorting-entrance.jpg

原文件：Screenshot_2024-02-07.png  
重命名：waste-sorting-detail.png
```

### 7. 发布前的检查清单

#### 内容检查：
- [ ] 文件名格式正确：YYYY-MM-DD-标题.md
- [ ] Frontmatter 信息完整
- [ ] 标签使用统一格式
- [ ] 图片路径正确
- [ ] 链接全部有效

#### 格式检查：
- [ ] 标题层级正确（H1, H2, H3）
- [ ] 列表格式统一
- [ ] 图片 alt 文本完整
- [ ] 引用格式正确

### 8. 常见问题解决

#### 模板不生效：
1. 检查插件是否开启
2. 确认模板文件夹路径正确
3. 重启 Obsidian

#### 图片路径错误：
1. 检查 images/observations/ 目录是否存在
2. 确认文件名无特殊字符
3. 使用相对路径

#### Git 冲突：
1. 拉取最新代码：`git pull origin main`
2. 解决冲突后重新提交
3. 推送到远程仓库
